# 封装一些经常用的函数
1. encodeStr($str) 将英文逗号分割的字符串转换成@包裹的字符串

| 参数   | 含义    |
|------|-------|
| str  | 1,2,3 |

1. decodeStr($str) 将英文逗号分割的@包裹的字符串转换成字符串

| 参数   | 含义                                       |
|------|------------------------------------------|
| str  | @1@,@2@,@3@                              |

1. cdnurl($url,$domain) 是否将图片换成cdn域名

| 参数      | 含义                         |
|---------|----------------------------|
| $url    | 文件半路径                      |
| $domain | true/false true==更换成为cdn域名 |
